# gamedev
Game engine course repo

# requirements
  * gcc 7.4.0
  * CMake 3.12.X

# instalation
on Ubuntu Linux:
  * git clone https://github.com/DmitryZapryvaryn/gamedev.git
  * cd gamedev
  * mkdir build && cd build
  * cmake ..
  * cmake --build .
