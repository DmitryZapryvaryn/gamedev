# gamedev
Game engine course repo

# requirements
  * gcc|g++ 8
  * CMake 3.15.X
  * SDL2 (https://www.libsdl.org/download-2.0.php)

# instalation
on Ubuntu Linux:
  * git clone https://github.com/DmitryZapryvaryn/gamedev.git
  * cd gamedev
  * mkdir build && cd build
  * cmake ..
  * cmake --build .
